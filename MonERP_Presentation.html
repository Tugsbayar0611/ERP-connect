💳 Цалингийн систем (Payroll)

Монголын хууль тогтоомжид нийцсэн автомат цалин тооцоолол

🧮 Татварын тооцоолол

  • НДШ (SHI) — Ажилтан 11.5%, Ажил олгогч 12.5%
  • ХХОАТ (PIT) — Шатлалт татвар (10-25%)
  • Доод цалин: ₮550,000 (2025)
  • Автомат тооцоолол — суурь цалин оруулахад бүгд тооцоологдоно
  • Татварын хөнгөлөлт тохиргоо
  • Ажил олгогчийн зардал тус тусад нь

📊 Ирцэд суурилсан тооцоо

  • Ажилласан өдрийн тооноос цалин тооцоолно
  • Өдрийн цалин = Суурь / Ажлын өдөр
  • Илүү цаг, урамшуулал нэмэх
  • Хагас сарын тохиргоо: 1-14, 15-сарын сүүл
  • Payroll Run: draft → processing → approved → paid
  • Олон салбар бүрт тусдаа Run

💵 Нэмэгдэл & Урьдчилгаа

  • Нэмэгдэл (Allowances) — Унаа, хоол, гэх мэт
  • Тогтмол/нэг удаагийн нэмэгдэл
  • Татвартай/татваргүй тохиргоо (isTaxable, isSHI, isPIT)
  • Хүчинтэй хугацаа: effectiveFrom → effectiveTo
  • Цалингийн урьдчилгаа → Батлах workflow
  • Зээлийн суутгал: сар бүр / нэг удаа / хүүтэй
  • Хоолны суутgal (Canteen wallet-ээс автомат)

📋 Payslip & Тайлан

  • Payslip (Цалингийн хуудас) — ажилтан бүрт
  • Earnings breakdown: суурь + нэмэгдлүүд
  • Deductions breakdown: НДШ + ХХОАТ + зээл
  • Олноор батлах / устгах (Bulk)
  • Pending → Processing → Paid статус
  • Цалингийн түүх, сарын тайлан
  • Dashboard-д сарын цалингийн зардлын график

🎯 Гүйцэтгэлийн удирдлага (Performance)

KPI зорилт тавих, хянах, үнэлэх, тайлагнах бүрэн workflow

📌 Үнэлгээний Үе (Period)

Хагас жил / Улирал / Жилийн зорилт. Active → Locked → Archived статус. Admin/Manager үүсгэнэ.

🎯 Зорилт (Goal)

Гарчиг, тайлбар, Metric Type (%, тоо, мөнгө, boolean), Зорилтот утга, Одоогийн утга, Жин (weight 0-100%), дуусах огноо.

📊 Оноо тооцоолол

Оноо = (Weight × Progress × Quality Rating) / 500. Quality Rating: 1-5 (Менежерийн үнэлгээ). Нийт жин 100% байх шаардлагатай.

📑 Evidence & Audit

Зорилт бүрт нотлох баримт хавсаргах. Бүх өөрчлөлтийн audit trail. Хэн, хэзээ, юу өөрчилсөн.

🔄 Workflow

Draft → Submitted → Approved → Evaluated → Locked. Ажилтан зорилт тавих → Менежер батлах → Гүйцэтгэл оруулах → Менежер үнэлэх.

👥 Багийн тайлан (Team Report)

Менежер бүх багийнхаа зорилтыг харах. Bulk Approve — олноор батлах. Хайлт & шүүлтүүр. Excel экспорт.

📥 Inbox систем

Менежерийн Inbox — батлах/үнэлэх зорилтууд автоматаар орж ирнэ. Статус бүрээр шүүх.

🔒 Period Lock & PDF

Үнэлгээний үе → Түгжих (Lock). Түгжсэний дараа өөрчлөлт хийх боломжгүй. PDF тайлан хэвлэх (jsPDF).

💰 Санхүү & Нягтлан бодох бүртгэл

Нэхэмжлэх, ерөнхий журнал, дансны төлөвлөгөө, банкны хуулга

🧾

Нэхэмжлэх (Invoices)

  • Худалдан авалт & борлуулалтын нэхэмжлэх
  • Автомат дугаарлалт (INV-2024-001)
  • Төлөлтийн хяналт: Draft → Posted → Paid / Overdue
  • PDF хэвлэх — компанийн лого, гарын үсэг
  • QPay QR → QR код үүсгэх → Төлбөр хүлээх
  • И-Баримт → PosAPI 3.0 автомат илгээх
  • Нэхэмжлэхийн мөрүүд: бараа, тоо, үнэ, НӨАТ
  • Повторны нэхэмжлэх (Recurring)
📒

Ерөнхий журнал (GL)

  • Double-entry — Дебит/Кредит бүртгэл
  • Posting Engine — Нэхэмжлэх → автомат GL бичилт
  • Журналын бичилт хайх, шүүх
  • Дансны үлдэгдэл (Account Balance)
  • Chart of Accounts — Дансны төлөвлөгөө CRUD
  • Дансны ангилал: Asset, Liability, Equity, Income, Expense
  • Бичилт бүрт тайлбар, огноо
🏦

Банк & Татвар

  • Банкны хуулга импорт (CSV/Excel)
  • Тулгалт (Reconciliation) — Хуулга ↔ GL
  • Татварын код тохиргоо (Tax Codes)
  • НӨАТ 10% стандарт, 0% чөлөөлөгдсөн
  • Татварын тайлан: НӨАТ, Excel/CSV экспорт
  • QPay callback → автомат GL posting

📊 Санхүүгийн тайлангууд

Стандарт санхүүгийн тайлангууд, график, экспорт

📋 Туршилтын үлдэгдэл (Trial Balance)

  • Бүх дансны Debit vs Credit
  • Огноо сонгох (эхлэх-дуусах)
  • Баланс зөв/зөрсөн автомат шалгалт ✅❌
  • Дансны код, нэр, үлдэгдэл

💼 Баланс (Balance Sheet)

  • Assets = Liabilities + Equity
  • 3 багана: Хөрөнгө / Өр / Өмч
  • Тайлангийн огноо сонгох
  • Баланс зөв эсэх автомат шалгалт

📈 Ашиг/Алдагдал (P&L)

  • Орлого vs Зарлага
  • Цэвэр ашиг/алдагдал
  • Recharts — Орлого vs Зардал Bar Chart
  • Зардлын бүтэц Pie Chart

🧾 НӨАТ тайлан (VAT)

  • Борлуулалт & Худалдан авалтын НӨАТ
  • Татварын суурь, хувь, дүн
  • Буруутгалгүй НӨАТ (Net Payable / Refundable)
  • Excel/CSV экспорт

📦 Агуулах & Бараа (Inventory)

Барааны бүртгэл, агуулахын нөөц, орлого/зарлага, хугацаа дуусах сэрэмжлүүлэг

📦

Барааны бүртгэл

  • SKU, нэр, тайлбар, баркод
  • Ангилал (Product Categories) — мод бүтэц
  • Төрөл: Бараа / Үйлчилгээ
  • Борлуулалтын үнэ / Өртөг үнэ
  • Нэгж: ширхэг, кг, литр
  • Нөөцийн хяналт: trackInventory / trackExpiry
  • Идэвхтэй/идэвхгүй статус
🏭

Агуулах & Нөөц

  • Олон агуулах (Warehouses) CRUD
  • Агуулах бүрт салбар хуваарилах
  • Үндсэн агуулах тэмдэглэх
  • Stock Levels — агуулах×бараа тоо хэмжээ
  • Reserved Quantity — захиалсан тоо
  • Нөөц дуусах сэрэмжлүүлэг
🔄

Орлого/Зарлага (Stock Movements)

  • 4 төрөл: In / Out / Adjustment / Transfer
  • Reference — захиалга, нэхэмжлэхтэй холбох
  • Batch Number — Баглааны дугаар
  • Expiry Date — Дуусах хугацаа
  • Тэмдэглэл, хэн бүртгэсэн
  • Dashboard-д 30 хоногийн сэрэмжлүүлэг
📊

Тайлан & Сэрэмжлүүлэг

  • Агуулахын нөөцийн нэгтгэл
  • Бараа бүрийн хөдөлгөөний түүх
  • Дуусах хугацааны сэрэмжлүүлэг (30 хоног)
  • Dashboard алданги widget
  • Low stock alert
  • Шүүлт: агуулах, ангилал, бараа

🛒 Борлуулалт & Худалдан авалт

Sales Order → Invoice → Payment → GL — бүрэн цикл

🛒 Борлуулалтын захиалга (Sales)

  • SO-2024-001 автомат дугаарлалт
  • Харилцагч (Customer) сонгох
  • Захиалгын мөрүүд: бараа, тоо, үнэ, хөнгөлөлт
  • Нийт дүн, НӨАТ тооцоолол
  • Статус: Draft → Confirmed → Delivered → Invoiced
  • Агуулахаас бараа хасалт автомат
  • Нэхэмжлэх автомат үүсгэх
  • Сарын борлуулалтын график (Dashboard)

📥 Худалдан авалт (Purchase)

  • PO-2024-001 автомат дугаарлалт
  • Бэлтгэн нийлүүлэгч (Supplier) сонгох
  • Захиалгын мөрүүд: бараа, тоо, өртөг
  • Статус: Draft → Ordered → Received → Invoiced
  • Агуулахад бараа нэмэлт автомат

🔒 Аюулгүй байдал & Хяналт

Системийн аюулгүй байдлыг олон давхаргаар хамгаална

🛡️

RBAC

Role-Based Access Control. Хэрэглэгч бүрт roles олгох. Roles → Permissions. Resource × Action тохиргоо.

🔐

2FA (Two-Factor)

Speakeasy + TOTP. Google Authenticator-тэй нийцтэй. QR кодоор тохируулах. Нэмэлт хамгаалалт.

📋

Audit Log

Бүх CRUD өөрчлөлтийг бүртгэнэ. Хэн, хэзээ, юу, хуучин/шинэ утга. IP, User Agent.

🔑

Session удирдлага

Идэвхтэй session хянах. Тус тусад нь устгах (revoke). IP, User Agent мэдээлэл.

🌐

Google OAuth

Passport.js + Google Strategy. Google-ээр нэвтрэх боломж. Tenant code-оор бүртгэл.

🔒

Helmet & Security

HTTP хамгаалалтын header-ууд. Rate limiting (brute-force). Bcrypt password hash. Password policy.

🏢 Multi-tenant архитектур

Нэг системд олон байгууллага — тус тусдаа мэдээлэл, тохиргоо

🔑 Tenant тусгаарлалт

  • Бүх хүснэгтэд tenantId — байгууллага тус тусдаа
  • Row-Level Security — нэг tenant нөгөөгийн мэдээлэл үзэх боломжгүй
  • Tenant бүрт: legalName, regNo, vatNo, address, city
  • Invite Code (MON-8899) — Шинэ хэрэглэгч бүртгүүлэх
  • Статус: active / suspended
  • Timezone: Asia/Ulaanbaatar
  • Currency: MNT

🏗️ Салбарын удирдлага (Branches)

  • Олон салбар: Толгой оффис (HQ) + Салбарууд
  • Салбар бүрт: нэр, код, хаяг
  • GPS координат: latitude, longitude
  • Geofence радиус: 50-500 метр
  • Оффис WiFi SSID тохируулах
  • Ажилтан → Салбар хуваарилах
  • Payroll Run → Салбар бүрт тусдаа

⚙️ Компанийн тохиргоо (Company Settings)

  • Ажлын цаг: эхлэх/дуусах (09:00 - 18:00)
  • Хоцрох threshold (минут)
  • Баримтын хандалтын бодлого: strict / history
  • Компанийн лого upload
  • QPay тохиргоо (client, secret)
  • И-Баримт тохиргоо (API endpoint, key)
  • Цаг агаарын тохиргоо (threshold-ууд)
  • SMTP email тохиргоо
