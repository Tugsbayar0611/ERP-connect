01. Mobile responsive тохиргоо

5 ажлын өдөр | 2026-05-06 - 2026-05-12

Гарах үр дүн: Гол хуудсууд 360px, 768px, 1366px+ дэлгэц дээр эвдрэлгүй харагдана.

  • Dashboard, sidebar, topbar, table, form, dialog, mobile navigation шалгах
  • Хэт өргөн хүснэгт, overflow, sticky action, input/button spacing засах
  • iOS/Android browser дээр touch target ба scroll behavior шалгах

02. UI сайжруулалт ба нэг төрлийн болгох

3 ажлын өдөр | 2026-05-13 - 2026-05-15

Гарах үр дүн: Захиалагчид харагдах үндсэн дэлгэцүүдийн polish, текст, empty/loading/error state цэгцэрнэ.

  • Card/table/form/button загварын ялгааг багасгах
  • Монгол label, validation, хоосон төлөв, алдааны мэдэгдлийг ойлгомжтой болгох
  • Critical workflow дээр action hierarchy, tooltip, badge, status өнгийг цэгцлэх

03. Ажиллагааны тест ба regression

5 ажлын өдөр | 2026-05-18 - 2026-05-22

Гарах үр дүн: Модуль бүрийн тестийн нотолгоо, issue log, UAT-д шилжүүлэх жагсаалт гарна.

  • Нэвтрэх, эрхийн түвшин, ажилтан, ирц, цалин, хүсэлт, агуулах, борлуулалт, тайлангийн урсгал шалгах
  • API, RBAC, database migration, импорт/экспорт, PDF/баримт хэвлэх урсгал шалгах
  • Critical/major/minor ангиллаар алдааг бүртгэж давтан шалгах

04. Алдаа засвар ба дахин баталгаажуулалт

4 ажлын өдөр | 2026-05-25 - 2026-05-28

Гарах үр дүн: Critical болон major алдаанууд хаагдаж, production-д шилжих go/no-go шийдвэр гарна.

  • Тестээс илэрсэн алдааг нөлөөллөөр нь эрэмбэлж засах
  • Засвар бүр дээр smoke/regression retest хийх
  • Нээлттэй үлдэх low priority item-уудыг post-launch backlog болгон тохирох

05. Production орчинд байршуулах

3 ажлын өдөр | 2026-05-29 - 2026-06-02

Гарах үр дүн: Production build, migration, SSL/domain, backup, rollback plan, handover checklist баталгаажна.

  • Production env, database, storage, email, external service key тохируулах
  • Build/deploy, smoke test, admin эрх, seed/initial data баталгаажуулах
  • Backup/rollback, monitoring, handover, богино хэрэглэгчийн заавар өгөх
