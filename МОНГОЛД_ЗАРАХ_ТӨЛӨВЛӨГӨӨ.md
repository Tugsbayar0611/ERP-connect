## 🎯 PHASE 1: ЯАРАЛТАЙ (1-2 долоо хоног)

### 1.1 Аюулгүй байдал 🔴
- [ ] **Нууц үгийн нарийвчлал** - Хүчтэй нууц үг шаардах
- [ ] **2FA (Two-Factor Authentication)** - SMS эсвэл Google Authenticator
- [ ] **Session timeout** - Идэвхгүй хэрэглэгчийг автомат гаргах
- [ ] **Rate limiting** - Brute force attack-аас хамгаалах
- [ ] **HTTPS enforce** - SSL certificate заавал шаардах
- [ ] **SQL injection хамгаалалт** - Input validation нэмэх

### 1.2 RBAC (Эрхийн систем) 🔴
```
Одоо: Хэрэглэгч бүр бүх зүйлд хандаж чадна
Хэрэгтэй: Role-based access control
```
- [ ] **Admin** - Бүх эрх
- [ ] **Нягтлан** - Санхүү, Тайлан
- [ ] **HR** - Ажилтан, Цалин, Ирц
- [ ] **Борлуулалт** - Sales, Invoice
- [ ] **Агуулахын ажилтан** - Inventory only
- [ ] **Viewer** - Зөвхөн харах

### 1.3 Монгол онцлог 🟡
- [ ] **РД (Регистрийн дугаар)** - Валидаци нэмэх
- [ ] **ХХОАТ тооцоолол** - Шатлал зөв тооцох
- [ ] **НДШ тооцоолол** - 11% ажилтан, 12.5% ажил олгогч
- [ ] **E-barimt интеграци** - Татварын албатай холбох (API)
- [ ] **Банкны интеграци** - Хаан, Голомт, ХХБ API
